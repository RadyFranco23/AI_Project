# 🚀 AI Projects Portfolio

Este repositorio reúne una colección de proyectos enfocados en **Inteligencia Artificial aplicada al desarrollo de software**, cubriendo áreas como:

* 🎙️ Procesamiento de voz (Speech-to-Text / Text-to-Speech)
* 🖼️ Computer Vision
* 🧠 Procesamiento de lenguaje natural (NLP)
* 🧪 Automatización de QA con IA
* 🏗️ Factoría de software basada en agentes

Cada proyecto demuestra la integración de servicios cloud (principalmente Azure) con soluciones prácticas orientadas a escenarios reales.

---

# 📂 Proyectos incluidos

## 🎙️ 1. Text-to-Speech con Azure AI

Convierte texto en voz utilizando Azure Cognitive Services.

**Highlights:**

* Voz neuronal realista
* Configuración con variables de entorno
* Ejecución en tiempo real

---

## 🎧 2. Speech-to-Text + Traducción

Pipeline completo de audio → texto → traducción.

**Tecnologías:** Whisper + Azure OpenAI

**Flujo:**

1. Captura de audio
2. Transcripción automática
3. Traducción a otro idioma

---

## 🖼️ 3. OCR con Azure Vision

Extracción de texto desde imágenes.

**Features:**

* Detección de texto por líneas y palabras
* Bounding boxes
* Procesamiento desde URL

---

## 🎯 4. Detección de Objetos (Azure + ONNX)

Sistema híbrido de detección de objetos en video.

**Capacidades:**

* Inferencia en cloud (Azure)
* Inferencia local (ONNX Runtime)
* Procesamiento de video en tiempo real
* Auto-etiquetado de dataset

---

## 📧 5. Detector de Spam (Machine Learning)

Clasificador de texto usando Naive Bayes.

**Pipeline:**

* Vectorización (Bag of Words)
* Entrenamiento supervisado
* Predicción en tiempo real

---

## 🧪 6. Generador de Casos de Prueba con IA

API que automatiza QA usando modelos de lenguaje.

**Arquitectura:**

* Generación de test cases
* Validación automática
* Corrección iterativa

---

## 🧠 7. Factoría de Software con Agentes de IA

Sistema avanzado que transforma requerimientos en artefactos de desarrollo.

**Capacidades:**

* Generación de grafos de dependencias
* Creación de historias de usuario
* Edición incremental del sistema
* Persistencia en Azure
